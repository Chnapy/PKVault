# 1 - Démarrage rapide

Un démarrage rapide de PKVault peut être fait en quelques étapes:

- Dans les paramètres, fournissez un ou plusieurs chemins vers vos sauvegardes
- Depuis la page du stockage, sélectionnez une de vos sauvegardes
- Le stockage de votre sauvegarde s'affiche alors
  - Sélectionnez une boîte du stockage
  - Sélectionnez un des pokémons
  - Déplacez le pokémon depuis la sauvegarde vers le stockage PKVault
  - Sauvegardez

A partir d'ici vous avez un bon point de départ pour utiliser PKVault !

Envisagez de parcourir le reste de la documentation pour plus d'information sur les fonctionnalités de l'application.
