# 2 - Session

Toutes les actions effectuées sont limitées à la session en cours.

Aucun changement n'est enregistré tant que la session n'est pas sauvegardée.
Les sauvegardes et fichiers ne sont pas modifiés.

Les actions peuvent être annulées avant sauvegarde.

Une fois sauvegardée, les actions de la session s'appliquent sur les fichiers (stockage, sauvegardes) et la session est alors re-créée. Un backup est toujours créé avant, ainsi il est toujours possible de revenir en arrière.
